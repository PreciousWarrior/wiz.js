# wiz.js
A library to interact with the wizemen API

# Installation
1. Install node.js and npm
2. Initialize your package via `npm init`
3. Run `npm i @preciouswarrior/wiz.js --save`
4. Create an `index.js` file, add the sample code, and run it via `node .`
5. Now code whatever you want!

[Documentation](https://github.com/PreciousWarrior/wiz.js/wiki)
